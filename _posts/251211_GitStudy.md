# 本文

こんにちは。山嵜です。

2025 年 12 月 10 日に、静岡県立大学草薙キャンパス 図書館 1F セミナールームにて Git や GitHub についての勉強会を開催しました。
私は冒頭で**バージョン管理の大切さ**についてプレゼンテーションをさせていただき、後に環境構築のサポートも行いました。

この勉強会では、基本的な Git の操作と GitHub の運用方法について理解してもらうことを目標としており、参加したすべての学生が環境構築を完了させ、GitHub にコミットを反映させるところまで達成することができました。

初めての勉強会でしたが、成功させることができました。
