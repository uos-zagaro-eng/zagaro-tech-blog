問 2

次のことを実施するUNIX のコマンドを答えなさい。

1.

ユーザー yuze のホームディレクトリにある"sample.txt"を、自分のホームディレクトリの prog というディレクトリにコピーする。
ただし、今のディレクトリ(カレントディレクトリ)は/(ルートディレクトリ)であるとする。ここで、カレントディレクトリとは、現在自分がいるディレクトリのことである。

模範解答

cp ~yuze/sample.txt ~/prog
2.

/bin/ 以下にあるファイルの一覧を、ls-bin.txtに保存する。

模範解答

ls /bin/ > ls-bin.txt

問 3

整数値を 2 つ入力させ、以下の式を計算して画面に出力するプログラムを作りたい。
ただし、整数値は順にx, yとする。

\frac{x + y}{2} + x^3 + \lfloor \sqrt{x} \log{10}{y} \rfloor

問われていること
  • math 関係の関数を覚えているか。
  • int / int が int 型になってしまう(小数点以下が表示されない)ことを回避しているか。
  • (x+y)を括弧で括れているか(計算順序が正しく行われているか)。

模範解答

#include<stdio.h>
#include<math.h>

int main(void){
    int x, y;
   
    //整数値を2つ入力させる
    scanf("%d %d", &x, &y);
    printf("%f\n", (x + y) / 2.0 + pow(x, 3) + floor( sqrt(x) * log10(y) ) );

    return 0;
}

問 4

ユークリッドの互除法を使って、2456と4596の最大公約数を求めなさい。
計算過程をすべて書くこと。

見たいポイント
  • ユークリッドの互除法のアルゴリズムを覚えているか。
  • それを正しい順番で使うことができているか。

模範解答
4596 = 2456 × 1 + 2140
2456 = 2140 × 1 + 316
2140 = 316×6+244
316 = 244×1+72
244 = 72×3+28
72 = 28 × 2 + 16
28 = 16 × 1 + 12
16=12×1 + 4
12 = 4×3
最大公約数は 4

問 5

実行すると、「キーボードから 100 個の整数値を入力すると、すべての値の入力後に、"3 の倍数番目と 5 の倍数番目に入力した値"を入力順に画面に表示し、さらに"画面に表示した値の和"を画面に表示する」プログラムを作りたい。

見たいポイント
  • 指定された内容を、指定した順番でできているか
  • 配列を使用できているか
  • 複数条件の if 文を正しく書くことができているか
  • 和の計算を行うことができるか。

模範解答

#include<stdio.h>

int main(void){
    int i, a[100];
    int sum = 0;

    for(i=0; i<100; i++){
        //a[]にキーボードから読み込んだ整数値を入れる.
        scanf("%d", &a[i]);
    }

    for(i=0; i<100; i++){
        //3の倍数または5の倍数を判別したいなら,
        //3または5で割ったときの剰余(あまり)が0であるかどうかを確認.
        if(i % 3 == 0 || i % 5 == 0){
            printf("%d\n", a[i]);
            sum += a[i]; //sum = sum + a[i]と同じ意味.
        }
    }

    //合計を表示.
    printf("%d\n", sum);

    return 0;
}

問 6

以下のプログラムを作成せよ。
「正の整数値をキーボードから 1 つ入力する(k とする)と、1 から 6 の間の整数値をランダムに k 個生成し、最後にそれぞれの値が何個生成されたかを表示する。」
ただし、1 回実行すると、1 から 6 の間のいずれかの整数値をランダムに 1 つ返す関数dice6()を作成して使用すること。また、乱数の種は、実行時の時刻によって変わるようにしなさい。

ポイント
  • プロトタイプ宣言の書き方
  • 関数の定義の書き方
  • 0 から 5 を出力する関数に対して、1 から 6 の場合でも正しく作れているか。
  • 乱数の使い方
    • 種を適切に設定できているか。

模範解答

#include<stdio.h>
#include<time.h>
#include<stdlib.h>

int dice6(void);

int main(void){
    srand((unsigned)time(NULL));
    int i, k;
    int count[7] = {0, 0, 0, 0, 0, 0, 0};

    //生成回数を入力
    scanf("%d", &k);

    for(i=0; i<k; i++){
        count[dice6()]++;
    }

    for(i=1; i<7; i++){
        printf("%dが出た回数は%d\n", i, count[i]);
    }

    return 0;
}


//1から6の値を返す関数
int dice6(void){
    return rand() % 6 + 1;
}
